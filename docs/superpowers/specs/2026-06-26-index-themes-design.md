# 首页主题扩展设计

## 目标
在原版 [index.html](../../index.html)（Bento / 玻璃质感）基础上，新增两个完全不同气质的首页主题，并保持原版不变，共三个版本供对比。

## 三个版本

1. **index.html（原版）**
   - 浅色、玻璃质感、Bento 卡片、分类筛选与搜索。
   - 不做改动，仅在新主题中加入指向它的切换链接。

2. **index-demo-tech.html（赛博霓虹科技风）**
   - **视觉**：深黑底、青紫霓虹渐变、网格扫描线、微光边框、故障文字。
   - **元素**：工具卡片改为"数据模块"，带发光 tag、数字编号、状态指示灯。
   - **动效**：背景轻微粒子/代码雨、卡片 hover 发光、标题 glitch 动画。
   - **交互**：保留分类筛选（改为 tab 式命令条）和搜索，支持打开与下载。

3. **index-demo-xianxia.html（水墨修真仙侠风）**
   - **视觉**：宣纸/墨色底、金色篆刻标题、竹青点缀、云雾粒子、细线边框。
   - **元素**：工具卡片改为"玉简/竹简"，带篆体编号、分类印章、毛笔字标题。
   - **动效**：云雾飘动、竹叶/花瓣飘落、卡片 hover 浮起与金光描边。
   - **交互**：保留分类筛选与搜索，支持打开与下载。

## 不变的内容
- 七个工具页面及其分类、描述。
- 打开与下载功能（`<a download>`）。
- 响应式布局与三个主题间互相跳转的 footer 链接。

## 实现方式
- 单文件 HTML + 内联 CSS/JS，无外部依赖，确保本地双击可用。
- 不使用真实图片，所有氛围通过 CSS 渐变、SVG 图案与 Canvas/JS 粒子完成。
- 性能友好：粒子数量受控，移动端降级或减少动画。
