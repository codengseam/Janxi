# 首页主题设计

## 目标
将多种首页风格合并到一个统一的 [index.html](../../index.html) 中，通过顶部皮肤切换按钮在三种风格间切换，并记住用户选择。

## 三种皮肤

1. **原版**
   - 浅色、玻璃质感、Bento 卡片、分类筛选与搜索。

2. **仙侠**
   - 水墨/墨色底、金色篆刻标题、竹青与朱砂点缀。
   - 工具卡片以“玉简”形式呈现，带篆体印章。
   - 背景有云雾与花瓣飘动的 Canvas 粒子效果。

3. **杂志**
   - 杂志/编辑排版风格，大号衬线标题、目录式列表与分类区块。

## 不变的内容
- 七个工具页面及其分类、描述。
- 打开与下载功能（`<a download>`）。
- 响应式布局。

## 实现方式
- 单文件 HTML + 内联 CSS/JS，无外部依赖，本地双击可用。
- 三种皮肤分别放在独立的 `<section class="skin">` 容器中，通过 `active` 类切换显示。
- 皮肤选择保存到 `localStorage`，刷新后自动恢复。
- 仙侠皮肤的 Canvas 动画仅在激活时绘制，控制性能开销。
