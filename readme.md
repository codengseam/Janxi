# 一句话短简介
简兮・轻量纯静态工具集 | 无后端依赖，即开即用的日常效率小工具
> 简兮简兮，方将万舞。——《诗经・邶风・简兮》
简兮（Janxi） 是一个纯静态页面构建的实用工具合集，取名自《诗经》「简兮」，承简约之旨，藏轻盈之态。
我们总在寻找趁手的小工具，却常常被臃肿的界面、繁杂的广告、不必要的注册打断节奏。简兮工具集，回归工具最本真的样子：纯静态页面，无后端依赖，不收集数据，打开就能用。
项目收录格式转码、番茄时钟、文本处理等各类日常高频小工具，所有工具均基于前端原生实现，无需后端服务，不上传用户数据，打开即可使用。
从格式转码到专注计时，从文本处理到日常计算，每一个小工具都精简克制，像散落在页间的细碎便利，随取随用，用完即走。
无扰，轻盈，简而有用。
